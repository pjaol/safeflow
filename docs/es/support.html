Preguntas frecuentes

¿Dónde se almacenan mis datos?

Todo se almacena localmente en tu iPhone. Clio Daye no tiene servidores y no realiza conexiones de red. Tus datos nunca salen de tu dispositivo.

¿Cómo elimino todos mis datos?

Ve a Configuración (el ícono de engranaje en la pantalla de inicio) y toca Eliminar todos mis datos. Esto elimina de forma inmediata y permanente todos los registros de ciclo, síntomas, entradas de estado de ánimo y notas. No se puede deshacer.

Olvidé mi PIN. ¿Cómo recupero el acceso?

Si Face ID o Touch ID está activado, úsalo para desbloquear la app. Si has perdido el acceso por completo, deberás eliminar y reinstalar la app — esto también eliminará tus datos, ya que no existe mecanismo de recuperación por diseño.

¿Puedo usar Clio Daye como método anticonceptivo?

No. Clio Daye es únicamente una herramienta de conciencia del ciclo. No la uses para anticoncepción o planificación familiar sin consultar a un profesional de salud.

¿Qué tan precisas son las predicciones?

Las predicciones se basan en el historial de ciclo registrado. Mejoran con más datos — al menos 3 ciclos completos proporcionan una línea base confiable. Clio Daye muestra rangos, no fechas exactas, para reflejar la variabilidad natural en la duración del ciclo.

¿La aplicación es de código abierto?

Sí. El código fuente completo está disponible en github.com/pjaol/safeflow. Puedes verificar directamente todas las declaraciones de privacidad.
