Questions fréquentes

Où sont stockées mes données ?

Tout est stocké localement sur votre iPhone. Clio Daye n'a pas de serveurs et n'établit aucune connexion réseau. Vos données ne quittent jamais votre appareil.

Comment supprimer toutes mes données ?

Allez dans Réglages (l'icône engrenage sur l'écran d'accueil) et appuyez sur Supprimer toutes mes données. Cela supprime immédiatement et définitivement tous les journaux de cycle, symptômes, entrées d'humeur et notes. Cette action est irréversible.

J'ai oublié mon code PIN. Comment puis-je récupérer l'accès ?

Si Face ID ou Touch ID est activé, utilisez-le pour déverrouiller l'application. Si vous n'avez plus accès du tout, vous devrez désinstaller et réinstaller l'application — cela supprimera également vos données, car il n'existe aucun mécanisme de récupération par conception.

Puis-je utiliser Clio Daye comme contraceptif ?

Non. Clio Daye est uniquement un outil de connaissance du cycle. Ne l'utilisez pas pour la contraception ou la planification familiale sans consulter un professionnel de santé.

Quelle est la précision des prédictions ?

Les prédictions sont basées sur l'historique de votre cycle enregistré. Elles s'améliorent avec plus de données — au moins 3 cycles complets donnent une base fiable. Clio Daye affiche des plages, pas des dates exactes, pour refléter la variabilité naturelle de la durée du cycle.

L'application est-elle open source ?

Oui. Le code source complet est disponible sur github.com/pjaol/safeflow. Vous pouvez vérifier directement toutes les déclarations de confidentialité.
